Carbon Footprint Tracker (Flask) - Final
- Custom tree absorption rate used: 15 kg CO2/tree/year
- How to run:
    1. Install requirements: pip install flask matplotlib
    2. Run: python app.py
    3. Open: http://127.0.0.1:5000
- To change tree absorption rate, edit ABSORB_RATE at top of app.py
- Database file (SQLite) will be created as carbon_final.db in the project folder.
